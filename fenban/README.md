# 学员自由组班小程序

这是一个纯前端网页小程序，直接打开 `index.html` 就能使用，也可以部署到主页同一站点下再通过链接或 `iframe` 接入。

## 功能

- 配置多个上课时间段
- 学员按时间段自由报名
- 每个时间段按报名顺序自动两两成班
- 单数学员自动显示为等待配对
- 管理员密码保护：添加、编辑、删除上课时间，移除班级或学员需要先登录管理员区
- 导出当前组班名单
- 数据保存在当前浏览器的 `localStorage`

## 管理员密码

首次打开页面时，左侧会出现“设置密码”。设置后，普通学员只能报名和查看组班结果；管理员输入密码后才能看到上课时间配置、编辑、删除和修改密码入口。

注意：这个版本是纯前端本地密码，适合先做演示和嵌入主页试用。真正上线给很多学员使用时，如果要严格防止别人绕过管理权限，需要接后端账号权限和数据库。

## 本地预览


可以直接双击打开：

```text

```

也可以用本地静态服务访问：

```text
http://127.0.0.1:4173/index.html
```

## 接入主页

方式一：在主页增加一个入口链接。

```html
<a href="/fenban/index.html">学员自由组班</a>
```

方式二：嵌入到主页的某个区域。

```html
<iframe
  src="/fenban/index.html"
  title="学员自由组班"
  style="width: 100%; min-height: 760px; border: 0;"
></iframe>
```

后续如果要让多个学员在不同手机/电脑上共享同一份数据，需要再接一个后端接口或数据库。
