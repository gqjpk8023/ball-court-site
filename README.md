# 国球精品刻乒乓培训官网

这是一个可以直接部署到 GitHub Pages 的静态官网，内容已接入学员自由组班小程序。

## 文件说明

- `index.html`：网站首页，包括球馆名称、介绍、价格、营业时间、联系方式和自由组班入口。
- `styles.css`：网站首页样式。
- `fenban/`：学员自由组班小程序。
- `assets/hero.jpg`：首页大图。
- `assets/entrance.jpg`：门头图。
- `assets/interior.jpg`：室内场地图。
- `assets/logo-square.png`：当前官网顶部 Logo。
- `assets/logo.png`：原长方形 Logo，保留用于备选。

## 组班小程序

访问地址：

```text
https://gqjpk8023.github.io/ball-court-site/fenban/
```

首次打开组班页面时，先设置管理员密码。普通学员只能报名和查看组班结果；管理员输入密码后可以添加、编辑、删除上课时间，并管理名单。

## 上传到 GitHub Pages

把这个文件夹里的所有内容上传到 `gqjpk8023/ball-court-site` 仓库根目录：

- `index.html`
- `styles.css`
- `README.md`
- `court-hero.png`
- `assets/`
- `fenban/`

上传完成后等待 GitHub Pages 自动刷新，一般 1 到 3 分钟生效。
